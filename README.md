# adwokat-trzebnica-html-to-php
 Na bazie załączonego szablonu strony w HTML pliku "index.html", stwórz 1:1 kopię taką samą stronę internetową jak z tego pliku HTML dla KANCELARIA ADWOKACKA ADWOKAT Katarzyna Maj. Twórz teksty bazując na pliku .docx, zamień wszystkie teksty w tym wygenerowanym nowym pliku strony dla kancelarii adekwatnie do tematyki strony. Teksty na stronie twórz pozycjonujące bardzo obszerne skupiając się na pozycjonowaniu SEO. Dodaj meta tagi Open Graph oraz Schema JSON. Struktura menu głównego strony: "Strona Główna", "O mnie", "Oferta" (Podmenu oferta: "Prawo cywilne", "Prawo spadkowe", "Obrona w sprawach karnych"), "FAQ & Pytania", "Kontakt". Na końcu refaktoryzuj HTML do PHP czystego żadnych frameworków. Zrób ze statycznych elementów w kodzie dynamiczne PHP $. Zrób template php include z sekcji <header></header> i zapisz jako "template-header.php". Zrób template php include z sekcji <head></head> i z możliwych statycznych elementów zrób dynamiczne $ i zapisz jako "template-head.php". Zrób template php include z sekcji <!-- Start footer section ============================================= --> do <!-- End  footer section ============================================= --> i zapisz jako "template-footer.php". Zrób template php include z sekcji <!-- For Js Library --> i zapisz jako "template-footer-scripts.php". Nanieś zmiany na pliki i zapisz stworzone zmiany w plikach w wygenerowanym w PHP pliku "index.php".
